두번째 강의 내용 요약

css의 탄생

  • 1.HTML의 등장 이후 웹의 인기가 엄청나게 높아졌고,인간의 욕심은 끝이 없기 떄문에 사람들은 웹페이지를 더욱 이쁘게 만들고 싶어 했다
  • 2.HTML에 디자인 태그를 추가해서 예쁘게 만들 수도 있겠지만
  • 3.그렇게 하는 것은 한계가 있기 떄문에 웹을 만든 사람들은 디자인을 위한 언어인 cascading Style Sheet,즉css라는 언어를 만들었다

    • css의 문법

  • 1.기본적으로 css코드는 이style이라고 하는 HTML 태그안에 적고 태그를 지정하라는 뜻으로 태그 이름을 쓰고,중괄호를 열고 그 안에 디자인을 바꾸는 코드를 작성하고 중괄호를 닫는 문법으로 되어있다.
  • 2.태그를 선택하는 부분을 선택자 라고 하고 중괄호에 있는 속성을 바꾸는 요소들을 가운데 콜론(:)을 기준으로 왼쪽은 속성명,오른쪽은 속성값이라고 한다.
  • 3.속성이 다 정의되었으면 항상 이 세미콜론(;)을 찍어서 속성이 끝났다는 것을 알려야 한다.
  • 4.HTML 태그 안에서도 직접 CSS 속성을 넣을 수가 있다.
  • 5.이 화면 처럼 태그에 style이라는 속성을 적고,따옴표안에 CSS 코드를 적으면 된다
  • 6.세미콜론(;)굳이 적을 필요는 없다.

    • css의 속성

  • 1.text-align:center;라고 적으면 글이 가운데로 정렬한다
  • 2.center위치에 left,right를 적으면 각각 왼쪽 오른쪽으로 이동한다
  • 3.여기서 알수있는것은 css에서 텍스트를 정렬하는 것은 이 text-align이라는 속성을 가지고 left,right,center라는 속성값을 가진다는 것을 알수있다
  • 4.css에서 텍스트의 크기를 바꾸는 것은 font-size라는 속성을 쓰면 되고 속성값은 어떤 단위를 넣으면 된다
  • 5.100px이라는 단위는,픽셀은 컴퓨터가 숫자를 세는 하나의 단위다
  • 6.font-family라는 속성을 넣고 그다음에는 따옴표를 하나 넣어야 된다 그러면 폰트가 달라진다

    • selector

  • a태그 선택지가 우선순위가 제일 낮고, class 선택자가 더 우선 순위가높고,id 선택자가 우선순위가 제일 높은것을 알수있다

    • box model

  • 1.시각적으로 h1태그는 화면 전체를 다쓰고 a테그는 자기 자신만큼 부피를 쓴다는 것을 알수있다
  • 2.화면 전체를 쓰는 요소들을 Block Level Element,자기자신 만큼의 부피를 쓰는 요소들을 Inline Level Element라고 한다
  • 3.Block Level과Inliine Level은 이태그들의 기본값일 뿐,display 속성을 언제든지 바꿀 수 있다
  • 4.그리고 이 태그들을 아예 안보이게 하려면 display:none;이라고 하면 이 태그들이 아예 사라진다
  • 5.Block Level Element의 크기를 바꿀려면 width속성을 이용하면 된다
  • 6.css가 적용되었는지 확인 하는 방법은 방법은 윈도우면 F12 혹은 ctrl+Shift+i macOS면 Option+command+i눌러 개발자 모드를 열고 확인할수있다

    • Flex box

  • 1.margin값을 없애면 box model이 붙는데 이를 나란히 놓을려면 부모태그가 필요하다 그러려면 아무 의미 없는 태그가 필요한데 그게 div 태그다
  • 2.display:flex;는 Flex기능을 활성화할수있다
  • 3.flex-direction: row;는 요소 수평 배치를 할수있다
  • 4.flex-direction: column;은 요소 수직 배치를 할수있다
  • 5.justify-content: flex-start;와justify-content:flex-end;는 Flex방향으로 요소 배치를 할수있다
  • 6.justify-content: center;는 요소 중앙 배치를 할수있다
  • 7.justify-content: space-around;는 간격을 두고 중앙 배치를 할수있다
  • 8.align-items: stretch; align-items: flex-start; align-items:flex-end는 flex 수직 방향으로 요소배치를 할수있다
  • 9.align-items: center;는 요소 중앙 배치를 할수있다
  • 10.flex-basis: 100px;는 flex 방향으로 요소 크기 변경을 할수있다
  • 11.flex-grow:1;는 여백 채우는 비율을 설정 할수있다
  • 12.flex-wrap:wrap;은 여백 부족 시 줄 바꿈 설정을 할수있다

    • Background Image

  • 1.vh라는 단위는 Vertical Height의 약자다
  • 2.div 태그의 display 기본값은 block이다
  • 3.filter속성을 사용해서 brightness를%값으로 밝기를 조절할수있다
  • 4.글의 그림자를 추가할려면 text-shadow 2px 2px 그리고 색상 코드로 추가할수있다

    • card Model

  • 1.text-align: left;를 입력해 카드 모델 안에서는 요소들이 왼쪽부터 작성되게 할수있다
  • 2.배경 색깔을 변경할려면 background-color를 입력하고 HEX코드를 입력해야 한다
  • 3.밑줄이 사라지게 할려면 text-decoration을none으로 입력하면된다
  • 4.img태그에 loading="lazy"속성을 넣으면 해당 이미지가 보일 때에만 로딩이 되게 해서 사이트의 용량을 줄일수 있다
  • 5.Inline Level Element가 필요할때 쓰라고 만든 아무의미 없는 태그가 있는데 그게 바로 span태그다
  • 6.card-container 전체를 복사해서 이렇게 하면 카드모델 여러 개가 추가된다
  • 7.card-container를 나란히 쓰면 카드들이 수평으로 배치되는데 .container를 나란히 쓰면 카드모델들이 세로로 배치된다

    • Media Query

  • Media Query의 문법은 우선'@'를 쓰고 media라고 쓴다 그다음 괄호를 열고 영어로 최대한 뜻의 Maximum을 줄인max-width 최대 크기가 800px인데 즉 800px미만인 모든 비율을 지정한다 그렇기 때문에 800px 이하가 되면 이div 태그의 border: none; 속성이 실행된다
