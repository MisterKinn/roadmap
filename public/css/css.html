About RoadMap

RoadMap 은 교내 유일 웹 개발 동아리이며,
웹 개발을 학습하고 함께 실습하며
웹 개발자로서의 역량을 쌓아나가는 동아리입니다.


여러분의 웹 사이트를 만들고 동아리 자체 서버와
도메인으로 웹을 호스팅하여 전 세계에서 여러분의 웹 페이지에
접속해 마법 같은 경험을 할 수 있도록 만듭니다.


또한 동아리 내부에서 개인·팀 프로젝트를 통해
리더십 및 협업 능력을 향상시키고,
해커톤과 같은 대회에 참가하여 부원들의
실력과 RoadMap의 명성을 높입니다.

Curriculum

웹 개발 기초
웹의 작동 방식과 웹의 기본 언어인
HTML, CSS 등을 학습합니다.
프론트엔드 기초

반응형 웹을 만듭니다.
프론트엔드 심화
React를 통해 더욱 효율적으로
웹을 개발합니다.
웹개발 마스터
Next.JS를 통해 대기업들이
실무에서 사용하는 방식으로
웹을 개발합니다.
